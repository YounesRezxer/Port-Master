# 🔥 Port-Master

**Port-Master** is a lightweight and fast TCP Port Scanner written in Python. It allows you to scan a target IP address, identify open and closed ports, and save the results to text files.

---

## ✨ Features

- 🚀 Fast TCP port scanning
- 🔍 Detects open ports
- ❌ Detects closed ports
- 💾 Saves scan results to text files
- 🖥 Simple and easy-to-use command-line interface
- ⚡ Great for learning networking and cybersecurity

---

## 🛠 Requirements

- Python 3

---

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/YounesRezxer/Port-Master.git
```

Navigate to the project directory:

```bash
cd Port-Master
```

---

## ▶️ Usage

Run the program:

```bash
python Port-Master.py
```

Enter the target IP address when prompted, and the scanner will begin scanning.

---

## 📂 Output

After the scan is complete, two files will be generated:

```
Open Ports[target].txt
Close Ports[target].txt
```

These files contain the lists of open and closed ports for the target.
